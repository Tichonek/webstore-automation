## Welcome to my automation tester portfolio!

This repository presents my projects and examples of automated tests created using Cypress, JavaScript, and auxiliary tools in Python.

## Reository contents

- cypress/e2e - E2E tests written in Cypress
- cypress/test_cases - Test cases for automation
- cypress/pages - Page Object Model objects
- cypress/scripts - support tools, e.g. test data generator
- cypress/scripts/tests - unit tests for scripts
- cypress/support - custom commands, config

## Technologies
- Cypress - E2E tests
- JavaScript - test logic
- Python - support tools

## How to run project

1. npm install
2. npx cypress open

## How to run python script
1. cd cypress/scripts
2. python -m main
